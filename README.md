# FireBaseInit
**Fichier d'initialisation de firebase (création des collections dans firestore)**

Importer la clé d'accés au service firestore dans la racine sous le nom "serviceAccountKey.json" puis lancer le service (node .\index)

## Récupérer une clé d'accés au service:

Firebase => Project_settings sideTab => Users_and_permissions tab => Advanced_permission_settings link (IAM & Admin) => Service_Accounts sideTab => Create_service_account button => create the account with OWNER permissions => select_account link (click name) => keys tab => Add_key button =>
